Modificar datos persoais

UMLUseCase
ESaude :: Use Case Model :: Casos de uso :: Modificar datos persoais

Description

Propósito: Modificar os datos persoais do usuario.

Pre-condición

O usuario visualizou os seus datos persoais

Escenario principal

  1. O sistema rexistra os cambios no perfil do usuario.

  2. Amósanse os datos modificados na ventá de datos do usuario

Escenario secundario

  • 2a: Os datos introducidos non son válidos.
    1. O sistema retornará á última versión válida dos datos dese usuario.
    2. Amósanse os datos antigos na ventá de datos do usuario.
    3. Móstrase unha mensaxe cos campos que produciron erro.

Post-condición

Modifícanse os datos do usuario no sistema
