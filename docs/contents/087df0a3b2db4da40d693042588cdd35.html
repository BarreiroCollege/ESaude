Aceptar-rexeitar cita

UMLUseCase
ESaude :: Use Case Model :: Casos de uso :: Doentes :: Aceptar-rexeitar cita

Description

Propósito: Aceptar ou rechazar a proposta horaria dunha cita

Pre-condición

O médico propón unha cita ao doente

Escenario principal

  1. Se se pulsa o botón “Aceptar” a cita quedará fixa na súa lista de citas ata que ocurra, e notifícase ao médico da conformidade do paciente.

  2. Se se pulsa “Rexeitar” a cita queda pendente dunha nova proposta horaria por parte do médico, a quen tamén se notificará.

Escenario secundario

  • 1a. Unha cita pendente de confirmación pasouse de fecha
    1. Rexeitase automaticamente para que o médico propoña unha nova data

Post-condición

A decisión é almacenada na táboa de citas do usuario e médico.
