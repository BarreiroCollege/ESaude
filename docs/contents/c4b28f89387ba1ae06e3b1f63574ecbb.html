Enviar feedback

UMLUseCase
ESaude :: Use Case Model :: Casos de uso :: Doentes :: Enviar feedback

Description

Propósito: Enviar unha mensaxe de feedback ao correo electrónico do administrador e a bandexa do sistema

Escenario principal

  1. Aparece a ventá de mensaxe na que se pode seleccionar o tipo de mensaxe (as queixas terán prioridade) e un corpo baleiro a completar ca mensaxe.

  2. Se se escribe algo no corpo, o botón de enviar habilitarase.

  3. Cando se pulse “Enviar” farase chegar a mensaxe ao correo electrónico do administrador así como unha copia á súa bandexa no sistema.

Post-condición

Mostrase por pantalla que o mensaxe enviouse correctamente.
