Rexistrarse

UMLUseCase
ESaude :: Use Case Model :: Casos de uso :: Usuarios anónimos :: Rexistrarse

Description

Escenario principal de éxito:

  1. O sistema amosa a ventá de rexistro.

  2. O usuario anónimo completa a información necesaria para rexistrarse (data de nacemento, xénero, alerxias, outras doenzas diagnosticadas, nome de usuario, contrasinal)

  3. O sistema autoriza o rexistro.

  4. O sistema confirma ao usuario anónimo que está rexistrado.

Escenario secundario:

  • 3a: Hai un erro nos datos do rexistro.
    .1: Indícanse os campos incorrectos para a súa corrección

Post-condición:

A información dun novo usuario é rexistrada no sistema.
